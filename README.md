# 美食推荐管理系统 java562

## 项目概述

美食推荐管理系统是一个集美食信息管理、用户互动于一体的综合性系统。系统基于SSM（Spring + SpringMVC + Mybatis）框架，采用JSP作为前端页面技术，致力于提供优质的美食推荐和管理服务。

## 技术栈

- 后端框架：Spring、SpringMVC、Mybatis
- 前端技术：JSP、JavaScript、CSS、jQuery

## 系统角色

- 管理员：负责整个系统的管理和运营。
- 用户：可以浏览美食信息并进行互动。

## 运行环境

- 开发工具：IDEA
- 运行环境：JDK、MySQL
- 项目构建：Maven

## 功能模块

### 管理员模块

- 个人中心
- 用户管理
- 美食分类管理
- 热门美食管理
- 美食教程管理
- 美食店铺管理
- 美食社区
- 系统管理
- 前台首页管理

### 用户模块

- 个人中心
- 美食教程管理
- 我的收藏管理
